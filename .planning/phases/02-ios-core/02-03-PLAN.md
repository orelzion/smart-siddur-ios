    First, add KosherSwift SPM dependency to the Xcode project:
    - Package: https://github.com/Elyahu41/KosherSwift (Swift port of KosherJava)

    **ZmanTime.swift**: Model for a single zman entry:
    - `id: String` (e.g., "alot", "netz", "sofZmanShma")
    - `name: String` (localized display name, e.g., "Alot HaShachar")
    - `hebrewName: String` (e.g., "עלות השחר")
    - `time: Date?` (nil if not applicable, e.g., polar regions)
    - `category: ZmanCategory` (enum: .dawn, .morning, .midday, .afternoon, .evening, .night)
    - `isEssential: Bool` (true for the ~8-10 default times, false for comprehensive-only)
    - `isNextUpcoming: Bool` (computed, set by view model)

    **ZmanimService.swift**: Wraps KosherSwift with opinion-aware calculation.
    - `func calculateZmanim(date: Date, location: UserLocation, opinions: ZmanimOpinions) -> [ZmanTime]`
    - `ZmanimOpinions` struct with: `dawnOpinion`, `sunriseOpinion`, `zmanOpinion`, `duskOpinion`, `shabbatCandleMinutes`, `shabbatEndMinutes`
    - Creates `GeoLocation` from user's lat/lng/elevation/timezone
    - Creates `ComplexZmanimCalendar` with that GeoLocation and the target date
    - Maps opinion settings to correct KosherSwift methods:
      - Dawn: alot_90 -> `.alos90`, alot_72 -> `.alos72`, alot_degrees -> `.alosHashachar` (16.1 degrees)
      - Sunrise: visible -> `.sunrise`, sea_level -> `.seaLevelSunrise`
      - Zman: mga -> MGA-based times, gra -> GRA-based times
      - Dusk: harav_ovadia -> `.tzaisGeonim3Point65Degrees`, gra -> `.tzais72`, baal_hatania -> `.tzaisBaalHatanya`, chazon_ish -> custom, rabenu_tam -> `.tzais72` (72 min)

    **Essential zmanim list** (~8-10, isEssential=true):
    1. Alot HaShachar (dawn, per opinion)
    2. Netz HaChama (sunrise, per opinion)
    3. Sof Zman Shma (per MGA/GRA opinion)
    4. Sof Zman Tfila (per MGA/GRA opinion)
    5. Chatzot (midday)
    6. Mincha Gedola
    7. Shkia (sunset)
    8. Tzeit HaKochavim (nightfall, per opinion)

    **Comprehensive additions** (~10 more, isEssential=false):
    9. Misheyakir (earliest tallit/tefillin)
    10. Sof Zman Shma (the other opinion -- MGA if default is GRA, vice versa)
    11. Sof Zman Tfila (other opinion)
    12. Mincha Ketana
    13. Plag HaMincha
    14. Candle lighting (Friday only, sunset - candleMinutes)
    15. Shkia (if not already essential based on opinion)
    16. Tzeit 72 min
    17. Tzeit Rabenu Tam
    18. Havdalah (Motzei Shabbat, sunset + endMinutes)
    19. Chatzot HaLaila (midnight)
    20. Alot (other opinion if not default)

    **Candle lighting / Havdalah**: Always include for Friday (candle lighting) and Saturday night (havdalah), regardless of essential/comprehensive toggle. Show in a separate "Shabbat Times" section.

    **ZmanimViewModel.swift**: `@Observable` class:
    - Receives `ZmanimService`, `SettingsRepository`, and `LocationRepository` from DependencyContainer (injected on init)
    - `zmanimList: [ZmanTime]` (all calculated zmanim for the date)
    - `showAllTimes: Bool` (toggle between essential and comprehensive, defaults to false)
    - `selectedDate: Date` (defaults to today)
    - `isShabbatToday: Bool`, `shabbatTimes: [ZmanTime]` (candle lighting / havdalah)
    - `nextZmanId: String?` (ID of the next upcoming zman)
    - On appear: calls `settingsRepository.fetchSyncedSettings()` and `locationRepository.getSelectedLocation()` to get current opinions and location, then passes them to `zmanimService.calculateZmanim()`
    - Recalculates when date, location, or opinions change
    - `markNextUpcoming()`: finds the first zman with time > now, marks it

    **ZmanimView.swift**: Main zmanim tab screen:
    - Header: Hebrew date, Gregorian date, location name
    - "Shabbat Times" card (shown on Friday/Shabbat): candle lighting time, havdalah time (styled distinctly)
    - Zmanim list: each row shows name + time, next upcoming zman highlighted with accent color / bold
    - "Show All Times" / "Essential Times" toggle button
    - Pull to refresh (recalculate)
    - Time format respects user's use24hFormat setting from LocalSettings

    **ZmanRowView.swift**: Single zman row component:
    - Left: zman name (Hebrew name as primary if language is Hebrew, else English)
    - Right: formatted time
    - Highlighted state: accent color background/text for next upcoming
    - Nil time handling: show "--:--" for unavailable zmanim

    **JewishDay.swift**: Model for a single day's Jewish calendar info:
    - `gregorianDate: Date`
    - `hebrewDay: Int`, `hebrewMonth: Int`, `hebrewYear: Int`
    - `hebrewDateString: String` (e.g., "י\"ב שבט תשפ\"ו")
    - `parsha: String?` (only on Shabbat)
    - `holiday: String?` (e.g., "Purim", "Pesach")
    - `isShabbat: Bool`
    - `isYomTov: Bool`
    - `isCholHamoed: Bool`
    - `isRoshChodesh: Bool`
    - `isTaanis: Bool` (fast day)
    - `omerDay: Int?` (1-49 during Sefirat HaOmer)
    - `dafYomi: String?` (e.g., "Bava Kamma 42")
    - `dayType: DayType` (enum: regular, shabbat, yomTov, cholHamoed, fastDay, roshChodesh) -- for calendar cell coloring

    **JewishCalendarService.swift**: Wraps KosherSwift JewishCalendar:
    - `func getJewishDay(for date: Date, isInIsrael: Bool) -> JewishDay`
      Creates JewishCalendar instance, queries all calendar state properties.
      - Holiday: `jewishCalendar.getYomTovIndex()` maps to holiday names
      - Parsha: `jewishCalendar.getParsha()` on Shabbat -> Parsha enum -> display name
      - Omer: `jewishCalendar.getDayOfOmer()` (returns 0 if not omer period)
      - Fast day: `jewishCalendar.isTaanis()`
      - Rosh Chodesh: `jewishCalendar.isRoshChodesh()`
      - Hebrew date: extract day/month/year from JewishCalendar
      - Daf Yomi: `YomiCalculator.getDafYomiBavli(jewishCalendar)` (if available in KosherSwift)
    - `func getJewishDaysForMonth(year: Int, month: Int, isInIsrael: Bool) -> [JewishDay]`
      Generate JewishDay for every day of the given Gregorian month. Used by calendar grid.

    **HebrewDateFormatter.swift**: Utility for formatting Hebrew dates:
    - Hebrew day numbers (gematria): 1=א, 2=ב, ..., 10=י, 11=י"א, 15=ט"ו, 16=ט"ז
    - Hebrew month names: תשרי, חשון, כסלו, טבת, שבט, אדר, ניסן, אייר, סיון, תמוז, אב, אלול
    - Handle Adar I / Adar II in leap years
    - Hebrew year (gematria of last 3 digits): e.g., 5786 -> תשפ"ו

    **CalendarViewModel.swift**: `@Observable` class:
    - `currentMonth: Date` (first day of displayed month)
    - `calendarMode: CalendarMode` (enum: .gregorianPrimary, .hebrewPrimary)
    - `daysInMonth: [JewishDay]` (all days for current month)
    - `selectedDay: JewishDay?` (tapped day, triggers detail sheet)
    - `showDayDetail: Bool` (sheet presentation toggle)
    - Navigate months: `goToNextMonth()`, `goToPreviousMonth()`, `goToToday()`
    - On month change: recalculate all JewishDay entries for the month
    - `monthTitle: String`: e.g., "February 2026" or "שבט תשפ\"ו" depending on mode
    - Loads isInIsrael from SettingsRepository for correct holiday calculations

    **CalendarGridView.swift**: The month grid component:
    - 7-column grid (Sun-Sat or custom day headers)
    - Each cell shows:
      - **Gregorian-primary mode**: Large Gregorian date number, small Hebrew date below
      - **Hebrew-primary mode**: Large Hebrew date (gematria), small Gregorian number below
    - Day markers: colored dots at bottom of cell:
      - Purple dot: Shabbat
      - Gold dot: Yom Tov / holiday
      - Red dot: Fast day
      - Blue dot: Rosh Chodesh
      - Green dot: Chol HaMoed
    - Today: circle/ring highlight
    - Tap: sets selectedDay and shows DayDetailSheet

    **CalendarView.swift**: Full calendar screen (Tab 2):
    - Month/Year header with left/right navigation arrows
    - "Today" button to jump to current month
    - Toggle button: "עברי" / "Gregorian" to switch calendarMode
    - CalendarGridView showing the month
    - Presents DayDetailSheet as .sheet when a day is tapped

    **DayDetailSheet.swift**: Bottom sheet shown on day tap:
    - Header: Full Hebrew date + Gregorian date
    - **Day Info section**:
      - Holiday name (if applicable, styled prominently)
      - Parsha (on Shabbat days)
      - Omer count (during Sefirat HaOmer): "Day X of the Omer"
      - Daf Yomi
      - Fast day indicator
      - Rosh Chodesh indicator
    - **Zmanim section**: Full zmanim list for this day (reuse ZmanimService.calculateZmanim)
      - Same essential list as ZmanimView
      - Include candle lighting / havdalah if Friday/Shabbat
